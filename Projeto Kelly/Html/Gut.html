GDE - O propósito do processo Gerência de Decisões é analisar possíveis decisões críticas usando um processo formal, com critérios estabelecidos, para avaliação das alternativas identificadas.


A Matriz GUT é uma ferramenta que auxilia na priorização de resolução de problemas. A análise GUT é muito utilizada naquelas questões em que é preciso de uma orientação para tomar decisões complexas e que exigem a análise de vários problemas. Para isso, com o sistema GUT é possível classificar cada problema de acordo com a Gravidade, Urgência e Tendência!
